Filters submitted emails and returns errors if they are from a free webmail service.

== Description ==

When an email field is submitted, this plugin looks at the submitted email and returns an error if the email address is from a free webmail service like Gmail, Yahoo, Hotmail, etc. Intended for forms where the email address needs to be a business email address.

Requires Formidable to be installed and activated first!

== Installation ==

1. Upload `formidable-no-webmails.php` to the `/wp-content/plugins/` directory
2. Activate the Formidable plugin through the 'Plugins' menu in WordPress
3. Activate this plugin through the 'Plugins' menu in WordPress
